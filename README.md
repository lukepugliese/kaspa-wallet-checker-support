# Kaspa Wallet Checker – Support

Kaspa Wallet Checker is a free, read-only lookup tool for the Kaspa (KAS) blockchain. It lets you check a public wallet address's balance, recent transactions, and the current network status.

This app does not create wallets, store private keys or seed phrases, hold funds, or let you send, receive, exchange, or trade cryptocurrency. It only reads publicly available blockchain data for addresses you enter.
